ConcurrentVersionsTree

class ConcurrentVersionsTree[A, B] extends ConcurrentVersions[A, B]

A ConcurrentVersions implementation that shall be used if updates modify current versioned values (= incremental updates). ConcurrentVersionsTree is a mutable data structure.

Please note: This implementation is not optimized yet and leaks memory at the moment (which shouldn't be a problem if the number of incremental updates is small). Also, future versions of ConcurrentVersionsTree will likely be immutable.
