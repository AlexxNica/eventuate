Getting started¶

To get a high-level overview of Eventuate, it is recommended to read sections Introduction and Architecture first. Optionally, you may also want to read some Articles about Eventuate. In a second step, clone the Eventuate Github repository and run the Example application.

Note

Eventuate requires an installation of Java 8 or later.

If you want to know more, read the User guide. It cover’s Eventuate’s main features and shows API usage examples. Although the user guide is a not a step-by-step tutorial, there’s enough information for creating your first Eventuate application.

If you have any questions or comments, please let us know.
